# 📊 Vrinda Store Data Analysis

## 📌 Project Overview

This project analyzes Vrinda Store sales data using Microsoft Excel to identify sales trends, customer behavior, and business performance through an interactive dashboard.

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting

## 📈 Dashboard Features

* Monthly Sales Analysis
* Sales by State
* Sales by Gender
* Order Status Analysis
* Sales Channel Performance
* Category-wise Sales
